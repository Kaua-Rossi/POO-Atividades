# Projeto de Gerenciamento Acadêmico

Um sistema CRUD CLI simples em Python para gerenciamento de campi e cursos de uma universidade, executado via terminal (CLI).

## Estrutura de Arquivos

O projeto é um monolito, composto por um único arquivo:

    main.py contém todo o código-fonte do projeto.

## Procedimentos para Executar o Código

Para executar este projeto, você precisa ter o Python 3 instalado em sua máquina.

    Salve o código em um arquivo com a extensão .py.

    Abra um terminal ou prompt de comando.

    Navegue até o diretório (pasta) onde você salvou o arquivo.

    Execute o script usando o seguinte comando:

  ```bash
  python3 main.py
  ```
OBS: Dependendo da configuração do seu sistema, o comando pode ser apenas python main.py.

O menu do sistema aparecerá no terminal, permitindo a interação.
